    Function sourceID

    Gets the library's version string (e.g. "3.8.7"), version number (e.g. 3_008_007) or source ID.

    string sourceID();

    These values are returned by the linked SQLite C library. They can be checked against the values of the enums defined by the d2sqlite3 package (SQLITE_VERSION, SQLITE_VERSION_NUMBER and SQLITE_SOURCE_ID).
